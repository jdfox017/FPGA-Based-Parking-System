FPGA-Based-Parking-System | School group project

Architected a digital control system using AHDL in Quartus II to manage a multi-floor parking garage, developing a system block diagram to map complex state-machine logic onto a Cyclone V FPGA board.
Executed hardware-level I/O mapping to bridge internal logic states to physical toggle switches and LED arrays for real-time tracking and visual data feedback.
